## :wrench: Setup

1. Clone this repository

```bash
$ git clone https://github.com/Emiri-i/rock-paper-scissors-game.git
```

2. Go into the repository

```bash
$ cd rock-paper-scissors-game
```

3. Install dependencies

```bash
$ npm install
```

4. Run the app

```bash
$ npm run start
```
